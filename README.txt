XploreJogja is a web based Management System Information application. 
The main function of this app is to see and manage whats going on in a tourism spot in Yogyakarta, Indonesia. 
So the manager can easily decide what action is needed to perform.

For example, the manager can see how much is the income and outcome.
Also, the manager can see how many tourists have come to the venue.

Output of the data can be see by either number or graphic.
There is also a statistical method that can show the total, max, min, and average of the data.
